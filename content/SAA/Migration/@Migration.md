1. **애플리케이션 전환**:

• 수정 없이 전환: **AWS MGN**.

• 서버 이미지 기반 전환: **AWS SMS**.

2. **데이터베이스 전환**:

• 데이터베이스 마이그레이션: **AWS DMS**.

3. **파일 및 스토리지 데이터 전환**:

• 동기화/복제: **AWS DataSync**.

• 물리적 전송: **AWS Snow Family**.

• 파일 전송 프로토콜 유지: **AWS Transfer Family**.

4. **중앙 관리 및 추적**:

• 모든 마이그레이션 작업 관리: **AWS Migration Hub**.
