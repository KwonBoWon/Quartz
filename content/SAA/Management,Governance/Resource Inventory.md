# Resource Inventory

## Resource Inventory 개념

- AWS 환경 내에서 현재 보유한 리소스(서버, 스토리지, IAM정책)등을 목록화하고 추적하는 과정
- 모든 리소스를 체계적으로 정리하는 것이 목적

### 관련 서비스

- Config: 리소스 변경사항 추적 및 기록
- Resource Groups: 리소스 논리적으로 그룹화 
- Systems Manager Inventory: EC2및 온프레미스 서버의 소프트웨어/구성 관리
- Trusted Advisor: 리소스 최적화 및 비용 절감 권장 사항 제공
