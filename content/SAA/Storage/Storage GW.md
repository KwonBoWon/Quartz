# Storage GW

## Storage GW 개념

- 온프레스 환경과 AWS 클라우드 스토리지를 연결(로컬 스토리지 처럼 사용 가능)
- File GW: 파일기반 워크로드 S3로 저장
	- S3 파일 게이트웨이
	- FSx파일 게이트웨이
- Volume GW: 블록 데이터(스냅샷 EBS로 저장)
	- Stored Volume GW: 모든 데이터를 온프레미스에서 로컬로 저장하고 S3에 백업하는 방식
	- Cache Volume GW:자주 액세스되는 데이터만 온프레미스 캐시 볼륨에 저장하고 나머지는 S3에 저장
- Tape GW: 가상 테이프 라이브러리 저장

## 사용사례

- 온프레미스 데이터 백업 및 마이그레이션
- 클라우드 기반 스토리지 확장
